CUSTOMIZATION:
--------------
Change Theme:
  Edit base/templates/base/base.html, line 3:
  <html lang="en" data-theme="efilwol">
  
  Try: "dark", "dracula", "forest"

Change Colors:
  Edit tailwind.config.js:
  efilwol: {
    "primary": "#8b5cf6",    // Your color here
    "secondary": "#ec4899",   // Your color here
    ...
  }
